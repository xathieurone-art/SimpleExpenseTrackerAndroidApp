📊 Simple Expense Tracker
A lightweight, intuitive Android application designed to help users monitor their daily spending and manage their budget effectively. Built with a focus on speed and simplicity.


🚀 Features
Quick Entry: Add expenses with a title, amount, and category in seconds.

Categorization: Organize spending into groups like Food, Transport, Rent, and Entertainment.

Transaction History: View a chronological list of all past expenses.

Monthly Overview: See a summary of total spending for the current month.

Data Persistence: Uses Room Database (or SQLite) to ensure your data stays on your device even after closing the app.

🛠️ Tech Stack
Language: Kotlin / Java

UI Framework: XML (View System) or Jetpack Compose

Database: Room Persistence Library

Architecture: MVVM (Model-View-ViewModel)

Local Storage: SharedPreferences (for user settings)
